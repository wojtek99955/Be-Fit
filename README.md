# Be Fit

## About

Fitness app

## Live

## Technologies

- reactjs
- typescript
- firebase
- formik
- styled-components
- yup
- react-charts
- react-charts-2
- framer-motion

## Features

- create user
- delete user
- change password, change name, change email
- add profile image
- forgot password via email
- tracking calorie intake
- fitness calculators
- weight calculators
- realtime database
- monthly statistics
- upload avatar images with cloud storage
- switch dark / light mode/
